# Vue Todo App

Simple Todo application built with Vue 3, Composition API, and Single File Components.

## Features

- ✅ Add / remove / toggle todos
- ✅ Edit todo items
- ✅ Filter todos (All / Active / Completed)
- ✅ Search todos
- ✅ Persist in `localStorage`
- ✅ Clean and responsive UI

## Project setup

```bash
npm install
npm run dev
```
