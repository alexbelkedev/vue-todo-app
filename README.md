# Vue Todo App

Simple Todo application built with Vue 3, Composition API, and Single File Components.

## Features

- ✅ Add / remove / toggle todos
- ✅ Persist in `localStorage`
- ✅ Clean UI

## Project setup

````bash
npm install
npm run dev

Then commit and push:
```bash
git add README.md
git commit -m "Add README"
git push
````
